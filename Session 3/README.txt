In the 'Documents Provided' directory, you can find the documents provided by the teaching staff, at the University of Nottingham, for this activity.
In the 'EEEBot_ESP32_firmware' directory, you can find the code (provided by the teaching staff) required to run the motors attached to the DOIT ESP32 DEVKIT V1, when provided with signals from the Arduino.
In the 'EEEBot_ArduinoNano_firmware' directory, you can find the code which the Arduino Nano runs for the main activity.
